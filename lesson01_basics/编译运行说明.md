# 第1课编译运行指南

## 方法一：使用CMake（推荐）

### Windows (Visual Studio)

```powershell
# 进入课程目录
cd lesson01_basics

# 创建build目录
mkdir build
cd build

# 生成Visual Studio项目
cmake .. -G "Visual Studio 17 2022"

# 编译
cmake --build . --config Release

# 运行
.\Release\lesson01.exe
```

### Windows (MinGW)

```powershell
cd lesson01_basics
mkdir build
cd build
cmake .. -G "MinGW Makefiles"
cmake --build .
.\lesson01.exe
```

### Linux/Mac

```bash
cd lesson01_basics
mkdir build
cd build
cmake ..
make
./lesson01
```

## 方法二：直接使用编译器

### Windows (g++)

```powershell
cd lesson01_basics
g++ -std=c++17 main.cpp -o lesson01.exe
.\lesson01.exe
```

### Linux/Mac

```bash
cd lesson01_basics
g++ -std=c++17 main.cpp -o lesson01
./lesson01
```

## 方法三：在Visual Studio中调试

1. 打开Visual Studio
2. 文件 -> 打开 -> CMakeLists.txt
3. 选择 `lesson01_basics/CMakeLists.txt`
4. 设置断点
5. 按F5开始调试

## 方法四：使用VS Code调试

创建 `.vscode/launch.json`:

```json
{
    "version": "0.2.0",
    "configurations": [
        {
            "name": "C++ Debug",
            "type": "cppdbg",
            "request": "launch",
            "program": "${workspaceFolder}/lesson01_basics/build/lesson01.exe",
            "args": [],
            "stopAtEntry": false,
            "cwd": "${workspaceFolder}/lesson01_basics",
            "environment": [],
            "externalConsole": false,
            "MIMode": "gdb",
            "miDebuggerPath": "gdb.exe",
            "setupCommands": [
                {
                    "description": "Enable pretty-printing for gdb",
                    "text": "-enable-pretty-printing",
                    "ignoreFailures": true
                }
            ],
            "preLaunchTask": "build"
        }
    ]
}
```

创建 `.vscode/tasks.json`:

```json
{
    "version": "2.0.0",
    "tasks": [
        {
            "label": "build",
            "type": "shell",
            "command": "cmake",
            "args": [
                "--build",
                "${workspaceFolder}/lesson01_basics/build",
                "--config",
                "Release"
            ],
            "group": {
                "kind": "build",
                "isDefault": true
            }
        }
    ]
}
```
